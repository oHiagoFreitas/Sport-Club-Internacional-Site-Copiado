# 🔴 Sport Club Internacional - Website Clone

Este projeto é uma recriação (clone) do site oficial do **Sport Club Internacional**, desenvolvida com foco em prática de front-end e aprimoramento de habilidades em HTML e CSS.

## 📌 Sobre o projeto

O objetivo deste projeto foi reproduzir o layout e a estrutura visual do site oficial do clube, trabalhando conceitos como:

* Estruturação semântica em HTML
* Estilização com CSS
* Flexbox para alinhamento e layout
* Organização de componentes (header, footer, seções)
* Boas práticas de UI

O site original pode ser acessado aqui:
👉 https://www.internacional.com.br ([Sport Club Internacional][1])

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* Flexbox

---

## 🎯 Funcionalidades

* ✅ Layout inspirado no site oficial
